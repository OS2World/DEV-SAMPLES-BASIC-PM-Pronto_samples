PRONTO.ZIP contains a demo for the product Pronto! PM, a development 
system that allows PM programs to be written in Microsoft BASIC.  The
ZIP file includes subdirectories from the demo diskette and a few 
duplicate file names, so it is important to unzip it with the switch 
that maintains internally stored directories 
(e.g., PKUNZIP2 -d PRONTO.ZIP).
